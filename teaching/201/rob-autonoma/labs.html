Laboratorio

Las sesiones de laboratorio utilizan Linux y Python.
  • Linux: Un tutorial rápido de líneas de comando se puede encontrar aquí. Se debe leer, al menos, los tutoriales 1, 2, 5 y 8.
  • Python: Para una introducción a Python y NumPy se puede clonar o descargar este repositorio, que tiene 2 partes: la primera es Python, y la segunda es NumPy.
El código que será utilizado a lo largo de todos los laboratorios de este curso, y que contiene los paquetes de ROS, se encuentra en: https://github.com/utecrobotics/labs_robautonoma_20201
